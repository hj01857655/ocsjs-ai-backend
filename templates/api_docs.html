📊 数据库监控接口

POST /api/db-monitor/test-connection 测试数据库连接

测试数据库连接状态，获取连接信息、版本和性能指标。

响应示例:

{ "success": true, "data": { "database_info": { "host": "interchange.proxy.rlwy.net", "port": 49225, "database": "railway", "user": "root" }, "connection_time": 45.67, "database_version": "8.0.35", "connection_status": "success" }, "message": "数据库连接测试成功" }
GET /api/db-monitor/stats 获取数据库统计

获取数据库连接池统计、查询统计和性能指标。

GET /api/db-monitor/health 数据库健康检查

检查数据库健康状态和连接质量。

🗄️ 表管理接口

GET /api/table-management/tables 获取所有表

获取数据库中所有表的列表和基本信息。

POST /api/table-management/query 执行SQL查询

执行自定义的SELECT查询语句。

请求参数:

sql (string, required)
SQL查询语句，只支持SELECT
limit (integer, optional)
结果限制数量，默认100，最大1000

💡 使用示例

// JavaScript 示例 // 1. 测试数据库连接 fetch('/api/db-monitor/test-connection', { method: 'POST', headers: { 'Content-Type': 'application/json' } }) .then(response => response.json()) .then(data => console.log(data)); // 2. 获取所有表 fetch('/api/table-management/tables') .then(response => response.json()) .then(data => console.log(data)); // 3. 执行查询 fetch('/api/table-management/query', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ sql: 'SELECT * FROM information_schema.tables LIMIT 5' }) }) .then(response => response.json()) .then(data => console.log(data));
